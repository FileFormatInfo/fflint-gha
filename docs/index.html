fflint Github Action

This is a Github Action that runs fflint (a file format linter) on your repository.

Parameters:

  • args - fflint args, either global or command-specific
  • command - which fflint command to run
  • files files to check
  • version - to force a specific version of fflint (or latest)

Example: YAML | results
